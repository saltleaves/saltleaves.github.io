跟说好的不一样呢

问题：右键菜单中没有 git bash 选项

  • 进入开始菜单找到 git bash ，然后通过 cd 进入相应目录执行命令。

问题：在github部署完成之后，马上访问可能出现404错误

  • 等待10分钟左右，查看邮箱是否有验证邮件

问题：配置连接git后，出现error deployer not found:github 的错误

  • 这个问题常常出现在hexo阶段命令行与本文不一样的人，可以尝试使用本文中的方法
